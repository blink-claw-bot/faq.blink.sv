# Which accounts can buy and sell bitcoin with Blink?

## **Location**

Currently the service is only available for people who have a bank account in El Salvador. \
\
The function is made available by country because bank transfers are tied to the service of specific banks.\
\
The extension to more countries (focusing first to Central America) is planned so please check back again if you are looking for a service in other countries.

## Account Level

To interact with the traditional banking system there is increased verification need so to be to Buy and Sell your Blink account needs to be upgraded to Level 2. See how to upgrade at [faq.blink.sv/using-blink/how-can-i-change-my-transaction-limits-on-my-blink-account](https://faq.blink.sv/using-blink/how-can-i-change-my-transaction-limits-on-my-blink-account)

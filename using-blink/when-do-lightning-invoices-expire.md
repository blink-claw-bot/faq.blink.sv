# When do lightning invoices expire?

The BOLT11 invoices generated by Blink expire after\
\* 2 hours using the Bitcoin account\
\* 5 minutes for the USD account.\
\
If an invoice has expired it cannot be paid.\
\
Other means to pay a Blink user are:\
\- Lightning Address: `username@blink.sv`\
\- static Paycode (also known as LNURLpay code) at `https://blink.sv/username`\
\- Point of Sale at `https://pay.blink.sv/username`\
\- an onchain bitcoin address for higher amounts

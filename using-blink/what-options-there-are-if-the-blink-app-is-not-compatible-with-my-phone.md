# What options there are if the Blink app is not compatible with my phone?

Phones running an Android version lower than 8.0 are no longer compatible with the Blink app.\
\
There are alternatives to connect to Blink:\
\
See [how to connect your Blink account to the Alby Browser extension on Android](https://app.gitbook.com/o/2ViMeR7UbzyiJOlhjTq1/s/62b2lq7ARWHvLbuaeYe9/\~/changes/33/integrations/connect-to-the-alby-browser-extension-on-android). \
\
You can also take similar steps to connect to Alby using a desktop browser like Chrome or Firefox.

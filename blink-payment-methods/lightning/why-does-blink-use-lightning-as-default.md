# Why does Blink use Lightning as default?

In addition to the inherent benefits of the Lightning Network, Blink chose to use Lightning by default because it aligns with our vision of making bitcoin accessible to all.&#x20;

By using Lightning, we can provide users with faster, cheaper, and more efficient transactions, making it easier to use bitcoin for everyday transactions.

Furthermore, by using the Lightning Network, Blink can help reduce the overall load on the Bitcoin blockchain, thus contributing to the overall health and sustainability of the Bitcoin network

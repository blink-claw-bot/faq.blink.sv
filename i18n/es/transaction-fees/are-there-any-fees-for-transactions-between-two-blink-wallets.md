# ¿Hay alguna tarifa por las transacciones entre dos billeteras de Blink?

No, no hay tarifas de transacción para enviar o recibir Bitcoin dentro de las billeteras de Blink.

Las transacciones entre dos billeteras de Blink se consideran internas y se procesan sin cargos adicionales. Sin embargo, al enviar Bitcoin a billeteras externas, pueden aplicarse tarifas regulares de red dependiendo del tipo de transacción (on-chain o relámpago).

# Tarifas de transacción

Las tarifas de transacción son una consideración importante al usar la billetera Blink para enviar o recibir pagos de Bitcoin. Las tarifas pueden variar según el método de transacción utilizado, ya sea intraledger, on-chain transacciones de la Red Lightning. En esta sección, cubriremos preguntas frecuentes sobre las tarifas de transacción, incluyendo cómo establecer tarifas, qué factores afectan las tarifas y cómo asegurarse de que su transacción se procese de manera eficiente manteniendo las tarifas bajas.

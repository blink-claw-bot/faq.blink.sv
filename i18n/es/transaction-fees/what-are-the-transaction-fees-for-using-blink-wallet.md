# ¿Cuáles son las tarifas de transacción para usar la billetera Blink?

Las tarifas de transacción de Blink están diseñadas para ser dinámicas y adaptarse a las condiciones actuales del mercado para proporcionar la mejor tarificación posible para nuestros usuarios. A continuación se muestra una tabla que muestra las tarifas promedio tanto para las transacciones en cadena como para las de la Red Lightning, así como las tarifas más bajas y más altas observadas. Tenga en cuenta que las tarifas pueden variar dependiendo de factores como la congestión de la red y las preferencias del usuario.

|                      | Depósitos                                               | Retiros                                                                                          |
|----------------------|--------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| **Intraledger**      | 0% tarifa de servicio                                         | 0% tarifa de servicio                                                                                      |
| **Lightning**        | 0% tarifa de servicio<br>+ El remitente paga las comisiones de enrutamiento de LN| 0% tarifa de servicio<br>+ comisiones de enrutamiento (~0.02% comisiones de enrutamiento históricas a monederos externos)              |
| **On-chain**         | **Para depósitos superiores a 1,000,000 sats:**<br>0% de comisión de servicio<br><br>**Para depósitos inferiores a 1,000,000 sats:**<br>5,000 sats de comisión fija | 10,000 sats tarifa fija<br>+ Comisiones de minería (varía según el mercado)<br>+ Por encima de 10M sats de desequilibrio LN/on-chain por mes, se aplican comisiones adicionales del 0.2% |
| **BTC <-> Conversión de Dólar** | $0 tarifa fija<br>+ 0.2% margen                                | $0 tarifa fija<br>+ 0.2% margen                                                                      |

Nota: Para mayor claridad, las tarifas enumeradas bajo cada tipo de transacción son acumulativas. Por ejemplo, si estás retirando a una dirección onchains, incurrirás en una tarifa fija de 10,000 sats + más la tarifa de minería (aplicable al mercado actual) + si la billetera tiene un [desequilibrio de más de 10M sats LN/on-chain](why-does-blink-have-a-lightning-onchain-imbalance-threshold.md) en un mes móvil: tarifas adicionales del 0.2%

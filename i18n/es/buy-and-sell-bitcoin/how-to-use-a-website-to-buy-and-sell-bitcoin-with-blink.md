# ¿Cómo usar un sitio web para comprar y vender bitcoin con Blink?

El sitio web [fiat.blink.sv](https://fiat.blink.sv) tiene la misma funcionalidad que el botón  `Comprar y vender bitcoin ` en la aplicación Blink.

Tu ID de cuenta es necesario para realizar un pedido. Esto se puede obtener de dos formas:

1. Copia el ID de cuenta desde la aplicación Blink
   1. Toque el menú en la parte superior derecha de la pantalla
   2. Toque \`Cuenta\`
   3. Copiar \`Tu ID de cuenta\` \
      ![](/.gitbook/assets/image.png)
2. Copia el ID de cuenta desde el panel de control de Blink.
   1. Iniciar sesión o crear una nueva cuenta con un número de teléfono en [dashboard.blink.sv](https://dashboard.blink.sv)
   2. Haz clic en el ícono de perfil en la parte superior derecha
   3.  Copia el `ID de cuenta de la ventana emergente`.

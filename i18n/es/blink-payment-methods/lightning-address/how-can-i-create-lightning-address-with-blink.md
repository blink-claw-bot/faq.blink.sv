# ¿Cómo puedo crear una dirección Lightning con Blink?

Tu dirección de Lightning en Blink se genera utilizando el nombre de usuario que creaste para tu cuenta de Blink. Si aún no has elegido un nombre de usuario, puedes seguir estas instrucciones para hacerlo.

* Haz clic en configuración
* Haz clic en "recibir pagos usando tu dirección de Lightning Address o la función de los comerciantes".
* Escribe tu nombre de usuario

Tu dirección de Lightning se verá así: username@blink.sv, donde \[username] es tu nombre de usuario personal de Blink que has establecido. Puedes compartir tu dirección de Lightning para recibir pagos de Lightning de otras billeteras compatibles.

Siguiendo estos pasos y configurando un nombre de usuario de Blink, también habrás desbloqueado las funciones de Caja Registradora y Código de Pago. La Caja Registradora te permite generar facturas para pagos de Lightning de manera rápida y sencilla. Tu Código de Pago único es un código QR estático que siempre permanece igual y al que puedes recibir pagos de Lightning. Puedes imprimirlo o compartirlo en redes sociales para recibir diferentes cantidades de pagos de Lightning, por ejemplo, propinas, donaciones u otros pagos.

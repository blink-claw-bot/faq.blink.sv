# Dirección Lightning y Paycode

La dirección Lightning proporciona direcciones legibles por humanos que se ven como una dirección de correo electrónico y a las que puedes enviar transacciones Lightning.

En esta sección de nuestras preguntas frecuentes, cubriremos todas las preguntas relacionadas con los pagos de Lightning Address y Paycode, incluyendo cómo utilizarlos, sus beneficios y cualquier consejo de solución de problemas que puedas necesitar.

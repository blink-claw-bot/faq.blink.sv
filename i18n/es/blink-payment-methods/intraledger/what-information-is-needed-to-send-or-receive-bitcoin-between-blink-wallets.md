# ¿Qué información se necesita para enviar o recibir Bitcoin entre billeteras Blink?

Para enviar Bitcoin a otro usuario de Blink, sólo necesitas saber su nombre de usuario de Blink. Simplemente introduce su nombre de usuario Blink como el destinatario, y los fondos serán enviados directamente a su billetera. Es una forma cómoda y sin complicaciones de realizar pagos dentro del ecosistema Blink.

Para recibir Bitcoin de otro usuario de Blink, necesitas proporcionarle tu nombre de usuario de Blink. Comparte tu nombre de usuario de Blink con el remitente, y podrá enviar fondos fácilmente a tu billetera utilizando tu nombre de usuario. Una vez que una transacción es completada con un usuario de Blink, su nombre de usuario será automáticamente guardado en tu lista de contactos, haciendo futuras transacciones aún más fáciles.

Con la sencilla interfaz de Blink, enviar y recibir Bitcoin entre billeteras Blink es fácil y sencillo. No se necesitan complicadas direcciones de billeteras ni códigos QR, lo que simplifica el proceso y mejora la experiencia del usuario.

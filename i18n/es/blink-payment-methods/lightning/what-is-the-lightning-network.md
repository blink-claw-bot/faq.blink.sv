# ¿Qué es la Red Lightning?

La Red Lightning es una solución de escalado de segunda capa construida sobre la red de Bitcoin, diseñada para permitir micropagos más rápidos y más baratos. Al utilizar la Red Lightning, los usuarios de Bitcoin pueden beneficiarse de una serie de ventajas, incluyendo:

1. **Bajas tarifas**: Las tarifas para procesar transacciones en la Red Lightning son extremadamente bajas, a menudo solo una fracción de un centavo. Esto contrasta con las redes de pago tradicionales, que pueden cobrar tarifas de hasta el 3-5% del monto de la transacción.
2. **La liquidación instantánea**: Las transacciones en la Red Lightning son finales, al igual que el efectivo. Esto elimina el riesgo de contracargos que enfrentan los comerciantes al aceptar pagos con tarjeta de crédito.
3. **Seguridad**: La Lightning Network se construye sobre la red de Bitcoin, por lo que se beneficia del mismo nivel de seguridad que la cadena de bloques subyacente.
4. **Escalabilidad**: La Lightning Network puede manejar un volumen mucho mayor de transacciones que la cadena de bloques de Bitcoin por sí sola, con el potencial de procesar millones de transacciones por segundo.

En general, la Lightning Network representa un avance significativo en la capacidad de Bitcoin para funcionar como una red de pago práctica y escalable.

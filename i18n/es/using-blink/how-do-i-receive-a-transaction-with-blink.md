# ¿Cómo recibo una transacción con Blink?

Blink fue creado para facilitar el envío y recepción de transacciones de Bitcoin utilizando on-chain, LNURL y Lightning.

Puede proporcionar la siguiente información para recibir una transacción:

* Tu dirección en la cadena (comienza con `bc1...`). Las transacciones entrantes en la cadena (depósito) requieren 2 confirmaciones para estar disponibles para gastar.
* Crear una factura de Lightning (formato BOLT11, comenzando con `lnbc1...`)
* Caja registradora aplicación web
* Utilice la Dirección Lightning, que utiliza una forma de dirección de correo electrónico (por ejemplo, yourusername@blink.sv)
* Usando tu Blink Paycode (también conocido como LNURL)

También puedes recibir pagos a tu nombre de usuario de Blink de otros usuarios de Blink. Ten en cuenta que esta función solo está disponible para pagos de Blink a Blink y NO funcionará con otras aplicaciones como Chivo o Strike.

# ¿Qué opciones hay si la aplicación Blink no es compatible con mi teléfono?

Los teléfonos que ejecutan una versión de Android inferior a 8.0 ya no son compatibles con la aplicación Blink.

Hay alternativas para conectarse a Blink:

Ver [cómo conectar tu cuenta de Blink a la extensión del navegador Alby en Android](../integrations/connect-to-the-alby-browser-extension-on-android.md).

También puedes seguir pasos similares para conectarte a Alby usando un navegador de escritorio como Chrome o Firefox.

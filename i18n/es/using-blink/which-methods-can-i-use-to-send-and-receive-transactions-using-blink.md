# ¿Qué métodos puedo usar para enviar y recibir transacciones usando Blink?

| Tipo de pago       | Enviando  | Recibiendo  |
| --------------------- | -------- | ---------- |
| Lightning             | Sí      | Sí        |
| On-Chain              | Sí      | Sí        |
| Dirección Lightning     | Sí      | Sí        |
| Lista de contactos\*        | Sí\*    | Sí\*      |
| Caja registradora aplicación web | No       | Sí        |
| LNURL Paycode         | No       | Sí        |

\*With the contact list you can send and receive to other Blink users only. and need to do at least one transaction with this user to add to “contact List”.

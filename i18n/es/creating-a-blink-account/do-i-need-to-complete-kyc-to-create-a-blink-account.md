# ¿Necesito completar KYC para crear una cuenta de Blink?

Todo lo que necesitas es un número de teléfono con permisos de mensajería de texto SMS internacionales y una conexión a internet.

Después de ingresar tu número de teléfono y recibir el mensaje de autenticación de cuenta por SMS o Whatsapp, puedes crear un nombre de usuario único para las transacciones internas de Blink y desbloquear otras funciones en la billetera.

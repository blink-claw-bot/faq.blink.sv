# ¿Qué debo hacer si no recibo un SMS?

Si no recibes el mensaje de texto SMS para completar tu registro en Blink, asegúrate de que tu teléfono tenga permisos de mensajería de texto SMS internacionales o esté conectado a internet para recibir el código de confirmación a través de WhatsApp.

Si aún no has recibido el código de confirmación después de tres intentos, por favor contacta al equipo de soporte de Blink para recibir asistencia directa.

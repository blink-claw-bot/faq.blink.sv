# ¿Cómo creo una cuenta con Blink?

Crear una cuenta de Blink es un proceso sencillo que implica los siguientes pasos:

1. Descarga la aplicación de Blink desde la app store de tu teléfono o [usa el APK en Android] (/creating-a-blink-account/how-to-install-the-apk-from-github.md)
2. Selecciona  `Configuración en la esquina superior derecha de la pantalla de inicio 
3. Seleccione la bandera según el código de área de tu número de teléfono.
4. Ingresa tu número de teléfono y desliza hacia la derecha para completar el rompecabezas.
5. Ingrese el código recibido por SMS.

## Agregar una dirección de correo electrónico (opcional)
Agregar una dirección de correo electrónico te permite usarla para iniciar sesión, por lo que no necesitas depender del número de teléfono. Pasos para agregar una dirección de correo electrónico:

1. selecciona el menú en la parte superior derecha y pulsa `Cuenta`
2. seleccionar `Email (para iniciar sesión)`
3. ingresa una dirección de correo electrónico y completa la verificación para habilitar el inicio de sesión con correo electrónico

## Establece tu nombre de usuario (opcional)
El nombre de usuario es tu identificador interno de blink y también sirve como tu [dirección de lightning](/blink-payment-methods/README.md)
6. Seleccione `Configuración`
7. Seleccione `Tu dirección de Blink` y haga clic en `Establecer tu dirección de Blink`
8. Elige un nombre de usuario para tu cuenta. Tu nombre de usuario no debe tener caracteres especiales ni espacios. Si tu nombre de usuario deseado no está disponible, puedes agregar números o más letras para verificar la disponibilidad.
9. Haz clic en `Establecer tu dirección de Blink` para finalizar la creación de tu cuenta.

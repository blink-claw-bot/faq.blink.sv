# Cómo instalar el APK de GitHub

Como la billetera Blink es software libre de código abierto, la aplicación se publica y se puede instalar desde las versiones en GitHub. La instalación de apps de terceros sólo está permitida en Android, no en iOS. Sigue estos pasos para instalar la aplicación desde GitHub:

1. Abre el enlace [github.com/GaloyMoney/galoy-mobile/releases/latest](https://github.com/GaloyMoney/galoy-mobile/releases/latest) en tu teléfono Android para descargar la última versión de la aplicación Blink.
2. Desplázate hacia abajo en la página hasta `Assets` y elige `app-universal-release.apk` para obtener la mejor compatibilidad.<br>
El `app-arm64-v8a-release.apk` es más pequeño pero podría no funcionar en dispositivos antiguos.
   ![](<../.gitbook/assets/apk_screenshot01.png>)
1. Confirm with `Download anyway` and wait for the download to finish\
   ![](../.gitbook/assets/apk_screenshot02.png)\
   ![](../.gitbook/assets/apk_screenshot03.png)\
   ![](../.gitbook/assets/apk_screenshot04.jpg)
3. Si aún no has configurado tu navegador para instalar aplicaciones de fuentes desconocidas, se te pedirá que lo hagas ahora. Elige `CONFIGURACIÓN` y habilita `Permitir desde esta fuente`.
   ![](../.gitbook/assets/apk_screenshot05.jpg)\
   ![](../.gitbook/assets/apk_screenshot06.jpg)\
4. Continúa con la instalación seleccionando `INSTALAR`
   ![](../.gitbook/assets/apk_screenshot07.jpg)\
   ![](../.gitbook/assets/apk_screenshot08.jpg)
5. Finally `OPEN` the app when installed\
   ![](../.gitbook/assets/apk_screenshot09.jpg)
6. Al igual que el resto de aplicaciones, también encontrarás la aplicación Blink en el cajón de aplicaciones o en el lanzador.

# ¿Qué información necesito proporcionar para registrarme en Blink?

Para crear una cuenta de Blink, necesitarás un número de teléfono con permisos de mensajería de texto SMS internacionales o WhatsApp y una conexión a internet.

Una vez que hayas ingresado tu número de teléfono, recibirás un mensaje de texto SMS o WhatsApp para la autenticación de la cuenta. Después de eso, puedes crear un nombre de usuario único opcional para las transacciones de Lightning Address e internas de Blink.

Si no tienes acceso a mensajería SMS, también ofrecemos WhatsApp como opción de verificación.

Tenga en cuenta que el nombre de usuario que cree será visible para otros usuarios de Blink, por lo que recomendamos elegir su nombre de usuario teniendo esto en cuenta.

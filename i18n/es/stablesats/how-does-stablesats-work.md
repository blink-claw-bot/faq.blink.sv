# ¿Cómo funciona Stablesats?

**Stablesats** funciona utilizando los mercados de derivados de Bitcoin para crear un dólar estadounidense sintético. Esto se hace a través del uso de un instrumento financiero llamado swap inverso perpetuo, que es esencialmente un contrato entre dos partes para intercambiar la diferencia en el valor de un activo (en este caso, Bitcoin) entre dos puntos en el tiempo.

Stablesats se crean al colateralizar Bitcoin y usarlo para abrir una posición perpetua de intercambio inverso. El intercambio inverso perpetuo está diseñado para mantener un valor de 1 dólar estadounidense, independientemente de las fluctuaciones en el precio de Bitcoin. Esto permite a los usuarios mantener, enviar y recibir dólares en sus billeteras Lightning sin la necesidad de stablecoins o integración con moneda fiduciaria.

En esencia, Stablesats proporciona una forma para que los usuarios de Lightning se protejan de las fluctuaciones en el tipo de cambio de Bitcoin y se aseguren de que puedan cumplir con sus obligaciones financieras sin tener que convertir constantemente entre Bitcoin y monedas fiduciarias.

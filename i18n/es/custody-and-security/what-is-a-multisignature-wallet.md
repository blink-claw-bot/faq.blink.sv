# ¿Qué es una billetera Multifirma?

**Multifirma** (multisig) es un mecanismo de seguridad que implica múltiples claves privadas para autorizar una transacción de Bitcoin. En lugar de una sola clave privada que controle la transacción, multisig requiere que dos o más claves privadas aprueben una transacción antes de que pueda ser ejecutada. Esto agrega una capa adicional de seguridad para prevenir el acceso no autorizado y el posible robo de fondos.

Si bien es cierto que los principios de Bitcoin enfatizan el control individual de las claves privadas, todavía existen razones válidas para usar un custodio como Blink. Las billeteras custodiadas pueden ofrecer medidas de seguridad adicionales como multisig, que pueden proporcionar una capa extra de protección para los fondos de los usuarios. Además, el uso de un custodio puede brindar tranquilidad a los usuarios que pueden tener menos experiencia en la gestión de sus propias claves privadas o que simplemente prefieren la comodidad y facilidad de uso que puede ofrecer una billetera custodiada. En última instancia, la elección entre la auto custodia y una billetera custodiada depende de las preferencias individuales y la tolerancia al riesgo.

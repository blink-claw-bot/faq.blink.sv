# ما هي الطرق التي يمكنني استخدامها لإرسال المعاملات واستلامها باستخدام Blink

| نوع الدفع       | ارسال  | تلقي  |
| --------------------- | -------- | ---------- |
| برق             | نعم      | نعم        |
| على السلسلة              | نعم      | نعم        |
| عنوان البرق     | نعم      | نعم        |
| قائمة جهات الاتصال*        | نعم*    | نعم*      |
| تطبيق الويب لنقاط البيع | لا       | نعم        |
| رمز الدفع LNURL         | لا       | نعم        |

* باستخدام قائمة جهات الاتصال ، يمكنك الإرسال والاستقبال إلى مستخدمي Blink الآخرين فقط. ستحتاج إلى إجراء معاملة واحدة على الأقل مع هذا المستخدم لإضافتها إلى "قائمة جهات الاتصال".

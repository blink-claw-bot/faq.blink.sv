# عنوان Lightning ورمز الدفع

يوفر Lightning Address عنوانا يمكن للبشر قراءته يشبه تماما عنوان بريد إلكتروني ويمكنك إرسال معاملات Lightning إليه.

في هذا القسم من الأسئلة الشائعة ، سنغطي جميع الأسئلة المتعلقة بمدفوعات Lightning Address و Paycode ، بما في ذلك كيفية استخدامها ومزاياها وأي نصائح لاستكشاف الأخطاء وإصلاحها قد تحتاجها.

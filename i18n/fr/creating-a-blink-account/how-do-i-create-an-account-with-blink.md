# Comment créer un compte avec Blink?

Créer un compte Blink est un processus simple qui comprend les étapes suivantes:

1. Téléchargez l’application Blink depuis la boutique d’applications de votre téléphone ou utilisez le fichier apk sur [Android] (/creating-a-blink-account/how-to-install-the-apk-from-github.md)
2. Sélectionnez `Paramètres dans le coin supérieur droit de l’écran d’accueil. 
3. Sélectionnez le drapeau correspondant à l’indicatif régional de votre numéro de téléphone
4. Entrez votre numéro de téléphone et faites glisser vers la droite pour compléter le puzzle
5. Entrez le code reçu par SMS

## Ajoutez une adresse e-mail (optionnel)
Ajouter une adresse e-mail vous permet de l’utiliser pour vous connecter, sans dépendre du numéro de téléphone. Voici les étapes pour ajouter une adresse e-mail:

1. sélectionnez le menu en haut à droite et appuyez sur `Compte`
2. sélectionnez `Email (pour la connexion)`
3. entrez une adresse e-mail et complétez la vérification pour activer la connexion par e-mail

## Définissez votre nom d’utilisateur (optionnel)
Le nom d’utilisateur est votre identifiant interne Blink et sert également [d’adresse Lightning] (/blink-payment-methods/README.md)
6. Sélectionnez `Paramètres`
7. Sélectionnez `Votre adresse Blink` puis cliquez sur `Définir votre adresse Blink`
8. Choisissez un nom d'utilisateur pour votre compte. Votre nom d'utilisateur ne doit contenir aucun caractère spécial ni espace. Si le nom d'utilisateur souhaité n'est pas disponible, vous pouvez ajouter des chiffres ou des lettres supplémentaires pour vérifier sa disponibilité.
9. Cliquez sur `Définir votre adresse Blink` pour finaliser la création de votre compte.

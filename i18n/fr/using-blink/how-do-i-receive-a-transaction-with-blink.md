# Comment puis-je recevoir une transaction avec Blink?

Blink a été créé pour faciliter l’envoi et la réception de transactions en bitcoin en utilisant on-chain, LNURL et Lightning.&#x20;

Vous pouvez fournir les informations suivantes pour recevoir une transaction:
  - Votre adresse on-chain (commence par `bc1...`). Les transactions entrantes on-chain (dépôts) nécessitent 2 confirmations pour être disponibles à la dépense.
  - Créer une facture Lightning (format BOLT11, commençant par `lnbc1...`)
  - Application web Point de Vente (PoS ou caisse enregistreuse)
  - Utilisez l’adresse Lightning, qui prend la forme d’une adresse e-mail (ex. : votrenomdutilisateur@blink.sv)
  - Utilisation de votre code de paiement Blink (également connu sous le nom de LNURL)

Vous pouvez également recevoir des paiements sur votre nom d’utilisateur Blink depuis d’autres utilisateurs Blink. Veuillez noter que cette fonctionnalité est uniquement disponible pour les paiements de Blink à Blink et ne fonctionnera PAS avec d’autres applications comme Chivo ou Strike.

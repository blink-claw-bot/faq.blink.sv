# Comment puis-je envoyer une transaction on-chain avec Blink?

Pour envoyer une transaction on-chain avec Blink, veuillez suivre les étapes suivantes:

1. Ouvrez l’application Blink et appuyez sur le bouton `Envoyer`.
2. Saisissez l’adresse Bitcoin on-chain du destinataire ou scannez son QR code.
3. Choisissez si vous souhaitez utiliser votre compte BTC ou votre compte Stablesats (USD).
4. Saisissez le montant que vous souhaitez envoyer ainsi qu’une note facultative pour vous-même.
5. Appuyez sur `Suivant` et vérifiez attentivement les informations.
6. Faites glisser le curseur vers la droite pour finaliser la transaction.
7. Une fois la transaction confirmée, vous pouvez partager l’identifiant de transaction (également appelé `hash` ou `txid`) avec le destinataire comme preuve de la réussite de la transaction.

Si les frais de minage calculés dépassent 10 % du montant que vous vous apprêtez à envoyer, Blink affichera une notification : `Frais élevés ! Nous recommandons Lightning.`

Veuillez noter que les transactions on-chain peuvent prendre un certain temps à être confirmées sur le réseau Bitcoin, en fonction du trafic actuel du réseau et des frais de transaction choisis.

# Comment puis-je envoyer une transaction avec Blink?

Blink propose un moyen simple d’envoyer des transactions en bitcoin via les réseaux on-chain et Lightning. Pour envoyer une transaction, vous pouvez procéder comme suit :
  - Saisissez l’adresse on-chain du destinataire (commençant par `bc1...`) ou une facture Lightning (commençant par `lnbc1...`).
  - Vous pouvez également utiliser l’adresse Lightning du destinataire, qui ressemble à une adresse e-mail (par exemple `nomutilisateur@blink.sv`), ou le nom d’utilisateur d’un autre utilisateur Blink pour effectuer le paiement.
  - Choisissez si vous souhaitez payer depuis votre compte Bitcoin ou votre compte Stablesats (USD).
  - Enfin, confirmez les détails de la transaction et envoyez le paiement.

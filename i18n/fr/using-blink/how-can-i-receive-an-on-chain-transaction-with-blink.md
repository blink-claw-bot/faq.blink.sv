# Comment puis-je recevoir une transaction on-chain avec Blink?

Pour recevoir une transaction on-chain avec Blink, suivez ces étapes:

1. Ouvrez l’application Blink et appuyez sur `Recevoir`.
2. Sous le QR code, cliquez sur `Onchain`.
3. Vous pouvez copier l’adresse, la partager ou montrer le QR code à l’expéditeur. Notez que, pour éviter la réutilisation d’adresses, une nouvelle adresse sera affichée après la réception d’une transaction sur l’adresse précédemment partagée. L’ancienne adresse restera néanmoins valide pour recevoir des transactions sur votre compte, même lorsqu’une nouvelle adresse est affichée.
4. Une fois la transaction envoyée, elle devrait apparaître sous peu comme `En attente` dans votre portefeuille. Cliquez sur la transaction pour voir son \`Hash de transaction\`. En utilisant ce hash de transaction (également appelé TXID), vous pouvez consulter les détails de la transaction dans un explorateur de blocs comme [mempool.space] (https://mempool.space) et voir quand elle est confirmée dans un bloc.

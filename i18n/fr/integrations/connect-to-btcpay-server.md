# Se connecter à BTCPay Server

Vous pouvez utiliser votre compte Blink pour recevoir et envoyer des paiements Lightning en ligne à l’aide d’un plugin BTCPay Server.

Annonce détaillée : [blink.sv/blog/introducing-the-blink-plugin-for-btcpay-server] (https://www.blink.sv/blog/introducing-the-blink-plugin-for-btcpay-server)

Documentation pour la connexion : [dev.blink.sv/examples/btcpayserver-plugin] (https://dev.blink.sv/examples/btcpayserver-plugin)

# Mes données de cercle ou mon nom d’utilisateur seront-ils publiés pour que d’autres puissent les voir?

Blink ne partagera votre nom d’utilisateur ou les statistiques de vos Cercles que si l’une des conditions suivantes est remplie:

* Vous partagez votre nom d’utilisateur et vos Cercles Blink publiquement sur les réseaux sociaux.
* Vous acceptez de participer aux classements où votre nom et vos statistiques seront affichés

# Engeri y'okukakasa nti yinivoyisi ya lightning yasasudwa?

Mu ttwale lya Lightning Network (LN), Obukakafu bw’okusasula (PoP) bukola ng’okukkiriza okukakasibwa nti enkolagana eyeetongodde ebaddewo bulungi. Okusasula kw’omulabe kwa atomu ekitegeeza nti singa wabaawo ekifaananyi ekisookerwako ekyafunibwa omusasula okusasula mazima ddala kwafunibwa node gy’egenda.

**Obukakafu Bw'okusasula Bukola Butya mu Lightning Network?**

1. **Okukola yinivoyisi nga tuyita mu Node y’oyo asasulwa**:
   * Enkola eno etandika n’okutondawo yinivoyisi ya Lightning Network nga ekolebwa node y’oyo asasulwa. Yinivoyisi eno si kusaba kwa kusasula kwokka kwokka wabula nsengeka nzibu erimu ebintu eby’enjawulo ebikulu ennyo mu nkolagana. Ekimu ku bintu ebikulu ebiri mu yinivoyisi eno ye cryptographic hash y’ekyama, emanyiddwa nga ‘payment pre-image’.
2. **Okussa omukono ku Yinivoyisi**:
   * Okukakasa obutuufu n’okutangira okukyusakyusa, node y’oyo asasulwa essa omukono ku yinivoyisi mu ngeri ya digito. Omukono guno bukakafu bwa cryptography nti ddala invoice yakolebwa nnannyini node yennyini so si mufere.
3. **Okusasula n'okubikkula Ekifaananyi ekisooka**:
   * Omusasuzi bw’amala okusalawo okutuukiriza yinivoyisi, okusasula kuyita mu Lightning Network okutuuka eri oyo asasulwa. Nga efunye obuwanguzi bw’okusasula, node y’omusasuzi efulumya ‘ekifaananyi ekisookerwako eky’okusasula’ eri oyo asasula. Ekifaananyi kino ekisookerwako mu bukulu kye kyama hash yaakyo eyateekebwa mu yinivoyisi.
4. **Okugatta Yinivoyisi ne'kifaananyi ekisookerwako nga obukakafu bw'okusasula**:
   * Omutendera ogusembayo mu kuteekawo Obukakafu bw’Okusasula kwe kugatta yinivoyisi ya LN eyasooka n’ekifaananyi ekisookerwako eky’okusasula. Omusasuzi asobola okukozesa ebitundu bino ebibiri okukakasa nti okusasula kwakolebwa eri oyo asasulwa. Mu bukulu, n’amawulire gano nti omuntu yenna asobola okukakasa nti hash eri mu yinivoyisi ekwatagana n’ekifaananyi ekisookerwako ekyaweebwa nga basasula, okukakasa nti ddala okutunda kwaliwo nga bwe kigambibwa.

**Lwaki obukakafu bw'okusasula bukulu?**

1. **Okukakasa Okusasula**:
   * PoP kikulu nnyo eri enjuyi okukakasa nti enkolagana eno yakolebwa bulungi nga tekyetaagisa kwesigama ku muntu wa kusatu.
2. **Obutagaana**:
   * Nga tulina obujulizi obw’ekyama obuweebwa PoP, oyo asasulwa tayinza kweegaana okufuna ensasula, era mu ngeri y’emu, oyo asasula tayinza kweegaana kusasula. Kino kikulu nnyo mu kugonjoola enkaayana n’okuzimba obwesige mu nkolagana ya digito.
3. **Obukuumi**:
   * Obutonde bwa cryptography obwa yinivoyisi n’ekifaananyi ekisookerwako bukakasa nti enkolagana eno ya bukuumi era nga tesobola kukyusibwakyusibwa oba kufera.

**Mu bufunze okukakasa nti yinivoyisi ya lightning yasasudwa waliwo amawulire ga'mirundi ebiri ageetaagisa:**

1.**Lightning Yinivoyisi** eyasooka ewereddwa node
2.**Payment Pre-image** efunibwa omusasuzi nga asasudde

Tukusaba bw'oba otuukirira ttiimu y'abakolera Blink oba abaddukanya node za lightning endala beera n'amawulire gano.

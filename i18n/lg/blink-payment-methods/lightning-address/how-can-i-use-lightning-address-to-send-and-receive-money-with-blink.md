# Nsobola ntya okukozesa Endagiriro ya Lightning okusindika n'okufuna ssente ne Blink?

Okusindika okusasula ku Ndagiriro ya Lightning:

1. Koppa Endagiriro yaabwe eya Lightning (okugeza nga. username@blink.sv)
2. Nyiga ku "Sindika" mu Blink ku lutimbe olusokerwaako
3. Teeka Endagiriro ya Lightning mu kitundu ky’elaga
4. Gattako ssente z'oyagala okusindika onyiga ku "Sindika"

Okufuna ensasula nga okozesa endagiriro ya Lightning:

1. Ggulawo menu y’okuteekawo waggulu ku ddyo ku lutimbe olusokerwaako olwa Blink
2. Funa "Endagiriro Yo Eya Blink"
3. Koppa Endagiriro yo eya Lightning (e.g. username@blink.sv) ng’onyiga ku yo
4. Gabana Lightning Address yo n'omuntu ayagala okukusasula\

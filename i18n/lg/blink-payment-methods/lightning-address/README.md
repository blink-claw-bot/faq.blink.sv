# Endagiriro ya Lightning ne Paycode

Endagiriro za Lightning ekuwa endagiriro esomebwa omuntu eringa endagiriro ya email era gy’osobola okuweereza enkolagana za Lightning.

Mu kitundu kino ekya FAQ yaffe, tujja kukwata ku bibuuzo byonna ebikwata ku kusasula Lightning Address ne Paycode, omuli engeri y’okubikozesaamu, emigaso gyabyo, n’obukodyo bwonna obw’okugonjoola ebizibu by’oyinza okwetaaga.

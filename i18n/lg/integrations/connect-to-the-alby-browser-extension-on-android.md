# Yunga ku Alby Browser extension ku Android

Mu kiseera kino endabika y'essimu ku Firefox ewagira eby’okwongerako ku Android yokka. Osobola okukola emitendera egy’engeri eno ku desktop nga kwotadde n’okukozesa Firefox, Chrome oba browser ekwatagana.

1. [Funa Firefox](https://www.mozilla.org/en-US/firefox/browsers/mobile/android/) Ku ssimu yo eya Android
2. Ggulawo Firefox era olonde `Add-ons` okuva mu menu waggulu ku ddyo

3. Kona ku `Find more add-ons` wansi

    <img src="/.gitbook/assets/alby_screenshot02.png" width="300">

4. Wandiika `Alby` mu kasanduuko k'okunoonya okuzuula ekyongereza

    <img src="/.gitbook/assets/alby_screenshot03.png" width="300">

    <img src="/.gitbook/assets/alby_screenshot04.png" width="300">

10. Goberera ekiragiro ku lutimbe - funa ekisumuluzo kya API mu tab oba eddirisa eppya. Funa ebisingawo ku bisumuluzo eby'oLutimbe ne API [wano](https://dev.blink.sv/api/auth).

    <img src="/.gitbook/assets/alby_screenshot10.png" width="300">

11. Bw’oteeka ekisumuluzo kya API n’oyungibwa bulungi akawunti yo eya Blink twala akaseera okuteeka akabonero ku ekyongereza ng’onyiga ku `Add` okumpi ne `Bookmarks`

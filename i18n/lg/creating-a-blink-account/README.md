# Okukola akawunti ya Blink

Mwaniriziddwa mu kitundu kya FAQ yaffe ekiweereddwayo okukola akawunti ne Blink. Wano, ojja kusanga eby’okuddamu mu bibuuzo byo byonna ebikwata ku kwewandiisa ku akawunti empya, okuteekawo eby’okwerinda, n’okutambulira mu nkola yaffe enyangu okukozesa. Oba oli mupya mu Bitcoin oba omukadde alina sizoni, tuli wano okukulambika mu nkola y’okukola akawunti ne Blink Wallet.

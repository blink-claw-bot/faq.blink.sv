# Ebisale by'oKuwaanyisa

Ebisale by'oKuwaanyisa kikulu nnyo ng'okozesa waleti ya Blink okusindika oba okufuna ssente za Bitcoin. Ebisale bisobola okwawukana okusinziira ku ngeri gye bikozesebwamu, ka bibe intraledger, on-chain oba okuwaanyisa ku mukutu gwa Lightning. Mu kitundu kino, tujja kwogera ku bibuuzo ebitera okubuuzibwa ebikwata ku ssente z'okutunda, omuli engeri y'okuteekawo ssente, ebintu ki ebikwata ku ssente, n'engeri y'okukakasa nti okuwaanyisa kw'okola kukolebwa bulungi ate nga ebisale bikuumibwa wansi.

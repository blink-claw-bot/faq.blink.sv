# Maapu y'abasuubuzi ya Blink

Mwaniriziddwa mu kitundu kya Maapu y'Abasuubuzi mu Blink FAQ. Mu kitundu kino, tujja kukwata ku bibuuzo ebya bulijjo ebikwata ku nkola ya Maapu y'Obusuubuzi esangibwa mu app ya Blink. Maapu y’Abasuubuzi ekusobozesa okwanguyirwa okufuna abasuubuzi mu kitundu kyo abakkiriza Bitcoin ng’enkola y’okusasula, ekikukwanguyira okukozesa Bitcoin yo mu nkolagana ya buli lunaku. Oba onoonya edduuka lya kaawa oba edduuka ly’engoye, Maapu y’Abasuubuzi ekubisse. Ka tugende mu bimu ku bibuuzo ebitera okubuuzibwa ku kitundu kino.

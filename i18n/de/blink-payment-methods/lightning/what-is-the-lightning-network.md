# Was ist das Lightning Netzwerk?

Das Lightning Network ist eine zweite Skalierungsebene, die auf dem Bitcoin-Netzwerk aufbaut und schnellere und günstigere Mikrozahlungen ermöglichen soll. Durch die Nutzung des Lightning Network können Bitcoin-Nutzer von einer Reihe von Vorteilen profitieren, darunter:

1. **Geringe Gebühren**: Die Gebühren für die Verarbeitung von Transaktionen über das Lightning Network sind extrem niedrig und betragen oft nur den Bruchteil eines Pennys. Dies steht im Gegensatz zu traditionellen Zahlungsnetzwerken, die Gebühren von bis zu 3-5% des Transaktionsbetrags verlangen können.
2. **Sofortige Abrechnung**: Transaktionen per Lightning Netzwerk sind endgültig, genau wie Bargeld. Damit entfällt das Risiko von Rückbuchungen, dem Händler bei der Annahme von Kreditkartenzahlungen ausgesetzt sind.
3. **Sicherheit**: Das Lightning Network baut auf dem Bitcoin-Netzwerk auf und profitiert daher von demselben Sicherheitsniveau wie die zugrunde liegende Blockchain.
4. **Skalierbarkeit**: Das Lightning Network kann ein viel höheres Transaktionsvolumen bewältigen als die Bitcoin-Blockchain allein und hat das Potenzial, Millionen von Transaktionen pro Sekunde zu verarbeiten.

Insgesamt stellt das Lightning Network einen bedeutenden Fortschritt in der Fähigkeit von Bitcoin dar, als praktisches und skalierbares Zahlungsnetzwerk zu funktionieren.

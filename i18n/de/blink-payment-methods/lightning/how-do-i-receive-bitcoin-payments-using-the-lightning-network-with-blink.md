# Wie erhalte ich Bitcoin-Zahlungen über das Lightning Network mit Blink?

Um eine Bitcoin-Zahlung mit Blink über das Lightning Netzwerk zu erhalten, folgen Sie einfach diesen Schritten:

1. Klicken Sie auf die Schaltfläche „Empfangen“ auf dem Hauptbildschirm der App.
2. Wählen Sie, ob Sie die Zahlung auf Ihr Bitcoin- oder Stablesats-Konto erhalten möchten
3. Wenn Sie einen bestimmten Betrag anfordern möchten, geben Sie den gewünschten Betrag in das entsprechende Feld ein.
4. Legen Sie eine Notiz oder ein Etikett an, um sich daran zu erinnern, wofür Sie Ihr Geld ausgegeben haben.
5. Sie können eine Rechnung mit einem Nullbetrag senden, aber das funktioniert nicht immer, je nachdem, welche Wallet der Absender verwendet. Einige Wallets wie Muun verlangen zum Beispiel immer eine Rechnung mit Betrag.

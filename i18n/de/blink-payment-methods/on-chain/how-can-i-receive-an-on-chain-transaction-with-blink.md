# Wie kann ich eine On Chain Transaktion mit Blink empfangen?

Um eine On Chain Transaktion mit Blink zu empfangen, gehen Sie folgendermaßen vor

1. Öffnen Sie die Blink-App und klicken Sie auf `Empfangen`.
2. Klicken Sie unter dem QR-Code auf `Onchain`
3. Sie können die Adresse kopieren, sie weitergeben oder dem Absender den QR-Code zeigen. Um die Wiederverwendung von Adressen zu vermeiden, wird eine neue Adresse angezeigt, wenn Sie eine Transaktion über die zuvor freigegebene Adresse erhalten haben. Die alte Adresse bleibt gültig, um Transaktionen auf Ihrem Konto zu empfangen, auch wenn eine neue Adresse angezeigt wird.
4. Sobald die Transaktion gesendet wurde, sollte sie in Ihrer Wallet kurz als `Pending` erscheinen. Klicken Sie auf die Transaktion, um ihren \`Transaktions-Hash\` zu sehen. Anhand des Transaktions-Hash (auch bekannt als TXID) können Sie die Details der Transaktion in einem Block-Explorer wie [mempool.space](https://mempool.space) einsehen und sehen, wann sie in einem Block bestätigt wird.

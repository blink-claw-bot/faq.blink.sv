# Was ist ein Zahlungscode und wie kann ich ihn verwenden?

Der Zahlungscode von Blink basiert auf dem LNURL-Protokoll. Er ermöglicht es Benutzern, einen statischen QR-Code zu erstellen, der für den Empfang von Lightning Zahlungen verwendet werden kann. Ihr einzigartiger QR-Zahlungscode kann auf einem Bildschirm angezeigt oder ausgedruckt werden, was ihn zu einer vielseitigen und bequemen Möglichkeit macht, Zahlungen in einer Vielzahl von Umgebungen zu akzeptieren.

Ihr Blink-Zahlungscode ist mit allen Lightning Wallets kompatibel, aber nur Lightning Wallets mit LNURL-Unterstützung können direkt damit bezahlen.

Nutzer von Lightning Wallets, die LNURL nicht unterstützen, müssen den Zahlungscode mit der Kamera ihres Mobiltelefons scannen und dem Link folgen. Sie werden zu Ihrem einzigartigen Point of Sale (PoS oder Kasse) weitergeleitet, wo sie eine Lightning-Rechnung erstellen können, um Sie zu bezahlen.

Zahler können den Betrag, den sie an Ihren Zahlungscode zahlen möchten, in ihrer Lightning Wallet festlegen und eine Beschreibung für die Zahlung hinzufügen.

So finden Sie Ihren individuellen Zahlungscode:

1. Klicken Sie auf das Menü oben rechts auf Ihrem Blink-Startbildschirm
2. Weiter zu „Wege zur Bezahlung“
3. Der Link zu Ihrem Zahlungscode wird hier angezeigt

# Unterstützen alle Wallets Lightning-Adressen?

Viele bekannte Wallets haben eine Unterstützung für Lightning Adressen hinzugefügt, aber es gibt immer noch einige, die sie nicht unterstützen.&#x20;

Ziemlich viele Lightning Wallets unterstützen das Senden an eine Lightning Adresse. Weniger Lightning Wallets können ihren Nutzern eine Lightning Adresse zur Verfügung stellen, um Zahlungen zu empfangen.&#x20;

Blink-Nutzer können beides tun, an Lightning-Adressen senden und Zahlungen von anderen Wallets empfangen, die das Senden an Lightning-Adressen unterstützen.

Einen Überblick über die verschiedenen Lightning Wallets und deren Unterstützung für Lightning Adressen finden Sie [hier](https://github.com/andrerfneves/lightning-address/blob/master/README.md#wallets-supported).

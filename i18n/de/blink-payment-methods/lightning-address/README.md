# Lightning Adresse und Zahlungscode

Die Lightning Adresse bietet eine von Menschen lesbare Adresse, die wie eine E-Mail-Adresse aussieht und an die Sie Lightning-Transaktionen senden können.

In diesem Abschnitt unserer FAQ behandeln wir alle Fragen im Zusammenhang mit Lightning Adressen und Zahlungscodes, einschließlich ihrer Verwendung, ihrer Vorteile und aller Tipps zur Fehlerbehebung, die Sie benötigen.

# Wie kann ich die Lightning Adresse verwenden, um mit Blink Geld zu senden und zu empfangen?

So senden Sie eine Zahlung an eine Lightning Adresse:

1. Kopieren Sie ihre Lightning-Adresse (z. B. username@blink.sv)
2. Klicken Sie auf „Senden“ in Ihrem Blink-Startbildschirm
3. Fügen Sie die Lightning Adresse in den Zielbereich ein
4. Fügen Sie den Betrag hinzu, den Sie senden möchten, und klicken Sie auf „Senden“.

Um Zahlungen per Lightning Adresse zu empfangen:

1. Öffnen Sie das Einstellungsmenü oben rechts auf Ihrem Blink-Startbildschirm
2. Suchen Sie „Ihre Blink Adresse“.
3. Kopieren Sie Ihre Lightning-Adresse (z. B. username@blink.sv), indem Sie sie anklicken.
4. Teilen Sie Ihre Lightning-Adresse mit der Person, die Sie bezahlen möchte.

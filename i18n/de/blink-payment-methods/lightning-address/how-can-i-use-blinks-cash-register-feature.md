# Wie kann ich die Point-of-Sale-Funktion (PoS oder Lightning Kasse) von Blink nutzen?

Es gibt zwei Möglichkeiten, den Blink Point of Sale zu nutzen:

**Option 1:**

1. Klicken Sie auf das Menü oben rechts auf Ihrem Blink-Startbildschirm
2. Klicken Sie auf „Zahlungsmöglichkeiten“.
3. Hier finden Sie Ihren PoS Link
4. Sie können diesen Link an Ihre Mitarbeiter und Kunden weitergeben oder ihn direkt in Ihre sozialen Medien, Ihre Website oder Ihren Blog einfügen, um Zahlungen zu erhalten.

**Option 2:**

1. Weiter zu [pay.blink.sv](https://pay.blink.sv/)
2. Geben Sie Ihren Blink-Benutzernamen ein oder melden Sie sich bei Blink an
3. Beginnen Sie mit dem Empfang von Zahlungen über Ihren einzigartigen Blink Point of Sale

Mit dem Point of Sale können Sie ganz einfach Zahlungen direkt auf beide Konten (BTC / USD) erhalten, ohne auf ein einziges Gerät angewiesen zu sein. Es ist eine ideale Lösung für Kleinunternehmer, die ein funktionierendes POS-System benötigen, ohne die zusätzlichen Kosten für traditionelle Zahlungsabwickler.

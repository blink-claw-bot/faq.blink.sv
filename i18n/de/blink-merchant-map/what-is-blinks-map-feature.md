# Was ist die Kartenfunktion von Blink?

Die Kartenfunktion in Blink ermöglicht es den Nutzern, einfach Unternehmen zu finden, die Bitcoin-Zahlungen akzeptieren. Durch den Zugriff auf die Kartenfunktion können Nutzer alle registrierten Unternehmen und deren Standorte einsehen.

Darüber hinaus ermöglicht die Kartenfunktion den Nutzern, Zahlungen direkt an einen Händler zu leisten, indem sie auf die Schaltfläche „An dieses Geschäft zahlen“ klicken, die erscheint, wenn ein Geschäftspin angeklickt wird. Dies vereinfacht den Zahlungsvorgang und macht ihn sowohl für Kunden als auch für Händler bequemer.

Wenn Sie ein Unternehmen sind, das Bitcoin-Zahlungen akzeptiert und in der Kartenfunktion aufgeführt werden möchte, wenden Sie sich einfach an den Support, um Anweisungen zur Registrierung Ihres Unternehmens zu erhalten.

Wenn Sie die Karte zum ersten Mal öffnen oder auf das Standort-Symbol klicken, können Sie Blink erlauben, auf den Standort Ihres Geräts zuzugreifen, so dass die Händlerkarte standardmäßig Ihren aktuellen Standort anzeigt.

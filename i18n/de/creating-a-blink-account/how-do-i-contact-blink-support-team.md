# Wie kann ich den Blink-Support kontaktieren?

Wenn Sie das Support-Team von Blink kontaktieren möchten, können Sie über die folgenden Kanäle:

* E-Mail: [support@blink.sv](mailto:support@blink.sv)

Sie können auch auf der [Blink-Statusseite] (https://blink.statuspage.io/) nachsehen, ob unser Team an aktuellen Vorfällen arbeitet.

# Blink Voucher

* Macht den Verkauf von Bitcoin für Fiat einfach
* Verkäufer können Provisionen verdienen
* Flexible Verkäuferprovisionen in der App eingestellt
* Keine Vorabkosten / Verpflichtung
* Ideal für Wechselstuben in bargeldbasierten Volkswirtschaften
* 65 Währungspaare und mehr
* Die Kosten sind: 1% Plattform-Gebühr für Blink
* Bezahlen Sie den Gutschein von Ihrem BTC- oder USD-Konto
* Optimiert für das Bitcoinize POS-Gerät
* Weitere kostengünstige Druckeroptionen sind geplant

### Ziele

* Mehr Peer-to-Peer-On- und Off-Ramps hervorbringen
* Bitcoin Akzeptiert -> Bitcoin Bevorzugt
* Eine neue Möglichkeit für Blink-Benutzer, Einnahmen zu erzielen

### Einen Gutschein einlösen

* Öffnen Sie [voucher.blink.sv] (http://voucher.blink.sv)
* Geben Sie den 12-stelligen Voucher-Code ein
* Sie können Geld mit Lightning-Wallets, die das LNURL-Abhebungsprotokoll unterstützen, oder durch On-Chain-Transaktionen abheben. Bitte beachten Sie jedoch, dass bei On-Chain-Transaktionen Transaktionsgebühren anfallen.

### Einen Voucher erstellen

* Öffnen Sie [voucher.blink.sv] (http://voucher.blink.sv)
* Melden Sie sich mit Ihren Blink-Zugangsdaten an oder erstellen Sie eine neue Wallet
* Geben Sie den USD-Betrag ein, den Sie verkaufen möchten
* Legen Sie den Prozentsatz der Provision fest, den Sie verdienen möchten
* Erstellen und finanzieren Sie den Voucher - wählen Sie, ob Sie von Ihrem BTC- oder USD-Guthaben auf dem verbundenen Blink-Konto bezahlen möchten
* Teilen Sie den QR-Code oder Gutscheincode oder drucken Sie ihn aus, wenn Sie ein Bitcoinize POS-Gerät oder einen anderen kompatiblen Drucker verwenden.
* Das Guthaben befindet sich nun auf dem Treuhandkonto und kann nur mit dem eindeutigen QR-Code oder dem Gutscheincode eingelöst werden.

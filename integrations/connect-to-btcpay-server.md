# Connect to BTCPay Server

You can use your Blink account to receive and send lightning payments online using a  BTCPay Server plugin.

Detailed announcement: [blink.sv/blog/introducing-the-blink-plugin-for-btcpay-server](https://www.blink.sv/blog/introducing-the-blink-plugin-for-btcpay-server)

Documentation on how connect: [dev.blink.sv/examples/btcpayserver-plugin](https://dev.blink.sv/examples/btcpayserver-plugin)
